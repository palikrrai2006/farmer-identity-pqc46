# Secure Farmer Identity Management Using Post-Quantum Cryptography & Blockchain

## Project Overview
Implementation of a quantum-safe farmer identity system for government subsidy programs.

**Problem:** ₹30,000-40,000 crore annual fraud in PM-KISAN subsidies
**Solution:** Biometric identity + Kyber KEM + Dilithium signing + Blockchain + IPFS
